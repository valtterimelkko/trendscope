# Velocity Brand Kit & Guide

> **The Bloomberg Terminal for Short-Form Video Trends**

---

## 01 Brand Heart

### Mission
Empower creators and agencies to detect trends before they peak—turning hours of manual scrolling into actionable intelligence delivered in real-time.

### Vision
Become the indispensable early-warning system for the creator economy, where speed and precision separate viral success from missed opportunity.

### Value Proposition
> **"Find the signal in the noise. Move while the window is open."**

Velocity detects trends in their first hours by monitoring velocity at the micro-influencer layer—alerting users within hours, not days, of a trend emerging.

### Brand Promise
- **Speed First:** Alerts within 6-24 hours of trend emergence
- **Action-Oriented:** Not just data, but intelligence you can act on
- **Professional Grade:** Enterprise-level precision at creator-friendly prices

**Rationale:**
- Directly communicates the core value proposition (speed of detection)
- Evokes momentum, acceleration, and forward movement
- Professional yet dynamic—appropriate for B2B SaaS

## 03 Brand Personality

### Archetype: The Smart Operator

A blend of **Smart Professor** (intelligence, expertise) and **Efficient Assistant** (reliability, action-oriented).

### Personality Attributes

| Attribute | Description | Voice Example |
|-----------|-------------|---------------|
| **Sharp** | Precise, incisive, no fluff | "Trend detected" not "We found something!" |

- ❌ Consumer-toy (not "fun" focused—we're a tool)

### Positioning Spectrum

## 04 Logo System

### Logo Concept

**Visual Metaphor:** A forward-leaning waveform or pulse line that suggests:
- Sound waves (audio trends)
- Velocity/acceleration curves
- Real-time data visualization
- Forward momentum

### Logo Variants

| Variant | Use Case | Minimum Size |
|---------|----------|--------------|
| **Primary Lockup** | Landing pages, marketing | 120px width |
| **Icon/Symbol** | Favicon, app icon, avatars | 16x16px |
| **Wordmark** | Co-branding, tight spaces | 80px width |
| **Reversed (White)** | Dark backgrounds | Same as primary |

### Logo Specifications

**Clear Space:** Maintain minimum padding equal to the height of the "V" in Velocity on all sides.

**Minimum Sizes:**
- Digital: 80px width (wordmark), 24px (icon)
- Print: 20mm width (wordmark), 6mm (icon)

**Logo Don'ts:**
- ❌ Don't stretch or distort
- ❌ Don't rotate
- ❌ Don't change colors outside brand palette
- ❌ Don't add effects (shadows, glows, bevels)
- ❌ Don't place on busy backgrounds

### Favicon Strategy
- **Format:** Multi-resolution ICO file (16x16, 32x32, 48x48)
- **Design:** Stylized "V" pulse icon
- **Test:** Must be visible on light, dark, and grey browser tabs

## 06 Typography

### Font Stack

**Primary Font:** Inter
- Modern, highly legible sans-serif
- Optimized for screen readability
- Extensive weight range (100-900)
- Free via Google Fonts

## 07 Voice & Tone

### Voice (Consistent Personality)

**Voice Attributes:**

1. **Direct** — Get to the point. No fluff.
   - ✓ "Trend detected in #beauty"
   - ✗ "Hey there! We noticed something interesting in the beauty category..."

2. **Confident** — State facts clearly. No hedging.
   - ✓ "Alert sent to Slack"
   - ✗ "Your alert should have been sent (we think)"

3. **Action-Oriented** — Focus on what user can do.
   - ✓ "Create video now"
   - ✗ "Trend information available"

4. **Professional** — Expert but not cold.
   - ✓ "Your weekly digest is ready"
   - ✗ "Yo fam, check this out! 🔥"

### Tone (Contextual Adaptation)

| Context | Tone | Example |
|---------|------|---------|
| **Onboarding** | Welcoming, guiding | "Welcome to Velocity. Let's set up your first alert." |
| **Success** | Confirming, encouraging | "Alert configured. You'll be the first to know." |
| **Error** | Helpful, clear | "We couldn't connect to Slack. Check your webhook URL." |
| **Alert** | Urgent, factual | "🔥 Trend Alert: Sound 'X' surging 340% in #beauty" |
| **Empty State** | Encouraging, guiding | "No trends detected yet. Add more niches to expand coverage." |

### Tone Spectrum by Situation

```
Neutral ◄──────────────────────────────► Urgent
   Welcome                    Trend Alert
   Settings                    Error
   
Standard ◄─────────────────────────────► Celebratory
   Confirmation              Onboarding complete
   Form submitted            Milestone reached
```

## 10 Motion & Animation

### Animation Principles
- **Purposeful:** Every animation guides attention or provides feedback
- **Fast:** 150-300ms duration for micro-interactions
- **Subtle:** Easing over linear, opacity/transform preferred

### Standard Transitions

| Animation | Duration | Easing | Use Case |
|-----------|----------|--------|----------|
| Button hover | 150ms | ease-out | Color/scale changes |
| Modal open | 200ms | ease-out | Scale + fade in |
| Page transition | 300ms | ease-in-out | Route changes |
| Loading pulse | 1500ms | ease-in-out | Infinite shimmer |
| Toast appear | 300ms | ease-out | Slide + fade |
| Trend indicator | 1000ms | linear | Pulsing "live" dot |

### Easing Functions
```css
--ease-out: cubic-bezier(0, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
```
