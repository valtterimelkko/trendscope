# Velocity Color Palette

> Complete color specifications for the Velocity brand

---

## Color Philosophy

Velocity's color palette reflects its core values:
- **Speed & Precision:** Clean blues suggest technology and trust
- **Professional Reliability:** Deep slate grays convey stability
- **Actionable Clarity:** Clear semantic colors for immediate comprehension
- **Modern Simplicity:** Restrained palette avoids visual noise

## Accessibility & Contrast

### WCAG AA Compliance

All color combinations meet WCAG AA standards (4.5:1 for text, 3:1 for UI):

| Foreground | Background | Ratio | Status |
|------------|------------|-------|--------|
| `#0F172A` | `#FFFFFF` | 15.8:1 | ✓ AAA |
| `#334155` | `#FFFFFF` | 8.6:1 | ✓ AAA |
| `#475569` | `#FFFFFF` | 6.0:1 | ✓ AA |
| `#3B82F6` | `#FFFFFF` | 3.9:1 | ✓ UI/Large Text |
| `#FFFFFF` | `#0F172A` | 15.8:1 | ✓ AAA |
| `#FFFFFF` | `#3B82F6` | 3.9:1 | ✓ UI/Large Text |
| `#FFFFFF` | `#10B981` | 4.5:1 | ✓ AA |
| `#FFFFFF` | `#EF4444` | 4.8:1 | ✓ AA |

### Color Blindness Considerations

- Never rely on color alone for critical information
- Pair colors with icons or text labels
- Success/Error states use different icons (check vs X)
- Trend velocity uses numeric percentages alongside colors

## Usage Guidelines

### Do's ✓
- Use Primary (`#0F172A`) for 80% of text
- Use Accent (`#3B82F6`) sparingly for maximum impact
- Use semantic colors consistently (green = good, red = bad)
- Maintain sufficient contrast ratios
- Test colors in dark mode

### Don'ts ✗
- Don't use Accent for body text
- Don't mix multiple semantic meanings for the same color
- Don't use pure black (`#000000`)
- Don't use colors outside this palette without approval
- Don't rely on color alone for accessibility
