## APPENDIX: COPY GUIDELINES

### Voice Checklist
- [ ] Sharp — Every word necessary? Cut 20%.
- [ ] Reliable — Promises we can keep?
- [ ] Fast — Can this be read in one breath?
- [ ] Professional — Would I say this to a respected colleague?
- [ ] Action-Oriented — Is the next step clear?

### CTA Best Practices
- Start with a verb
- 1-3 words maximum
- "Now" adds urgency for time-sensitive actions
- Use arrows (→) to indicate forward movement

### Word Choices
| Use This | Not This |
|----------|----------|
| Alert | Notification |
| Detected | Found / Discovered |
| Saturation | Popularity / Full |
| Velocity | Speed / Growth rate |
| Niche | Category / Topic |
| Window | Opportunity / Time |
| Monitor | Watch / Track |
| Configure | Set up / Change |
