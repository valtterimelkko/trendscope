## Executive Summary

This SEO content strategy establishes Trendscope as the definitive authority in TikTok trend intelligence through a pillar-cluster architecture. The strategy targets creators, agencies, and brand teams at every funnel stage—building organic visibility around high-intent keywords while establishing thought leadership in the emerging "trend newsroom" category.

**Strategic Approach:**
- **4 Content Pillars** covering core user intents
- **32 Cluster Pieces** supporting pillar authority
- **Funnel-Aligned Distribution** balancing education and conversion
- **Internal Linking Architecture** maximizing SEO value transfer

**Primary KPIs:**
- Organic traffic growth: 10,000+ monthly visits by Month 6
- Keyword rankings: Top 10 for 25+ target keywords
- Conversion rate: 3%+ visitor-to-trial rate on BoFu content

## 3. Internal Linking Architecture

### Link Hierarchy

```
                    Homepage
                        |
    +-------------------+-------------------+
    |                   |                   |
Pillar 1            Pillar 2            Pillar 3            Pillar 4
(Trend              (Productivity)      (Analytics)         (Agency)
Detection)              |                   |                   |
    |                   |                   |                   |
Clusters 1.1-1.8   Clusters 2.1-2.8   Clusters 3.1-3.8   Clusters 4.1-4.8
    |                   |                   |                   |
    +-------------------+-------------------+-------------------+
                        |
                Product/Features Pages
                        |
                    Pricing/Trial
```

### Linking Rules

1. **Pillar → Clusters:** Each pillar links to all its cluster pieces in a "Related Guides" section
2. **Clusters → Pillar:** Every cluster piece has a contextual link to its pillar within first 300 words
3. **Cross-Pillar Links:** Related pillars link to each other (e.g., Pillar 1 ↔ Pillar 3)
4. **BoFu Priority:** Bottom-funnel content gets prominent CTA links to pricing/trial
5. **Contextual Only:** No navigation-style link lists; all links within relevant content

### Anchor Text Strategy

| Link Type | Anchor Pattern | Example |
|-----------|----------------|---------|
| Pillar Links | Descriptive phrase | "complete guide to TikTok trend detection" |
| Product Links | Solution-focused | "trend alert system", "velocity tracking" |
| Trial Links | Action-oriented | "start free trial", "get early access" |
| Related Content | Topic + format | "our analysis of trend decay patterns" |

## 8. Content Production Guidelines

### Brand Voice Application

**For ToFu Content:**
- Professional but accessible
- Data-driven without being dry
- Problem-empathetic, solution-confident
- Use "creator" and "agency" language authentically

**For MoFu Content:**
- Sharper, more prescriptive
- Feature-aware without being salesy
- Action-oriented frameworks
- "Here's exactly how to do this" tone

**For BoFu Content:**
- Direct and conversion-focused
- Transparent about differentiators
- ROI and value-focused
- Confidence without arrogance

### SEO Technical Requirements

**Every piece must include:**
- [ ] Title tag: 50-60 characters, keyword front-loaded
- [ ] Meta description: 150-160 characters, CTA included
- [ ] H1: One per page, matches title intent
- [ ] H2-H6: Logical hierarchy, keyword variations
- [ ] Alt text: Descriptive, keyword where natural
- [ ] Internal links: Minimum 3 per 1,000 words
- [ ] Schema markup: Article or Guide type
- [ ] URL: Short, keyword-focused, hyphens

**Content freshness:**
- Update top 10% performers monthly
- Refresh data/statistics quarterly
- Review and improve underperformers every 90 days

### Content Distribution

**Publish → Amplify Sequence:**
1. **Day 1:** Publish with on-page SEO
2. **Day 2:** Newsletter to subscribers
3. **Day 3:** Social snippets (3-5 posts per piece)
4. **Day 5:** Submit to relevant aggregators (HackerNews, Reddit communities)
5. **Week 2:** Repurpose into Twitter/X thread
6. **Week 3:** Create short-form video summary
7. **Month 2:** Update and reshare best performers
